ul tag

ul tag means unordered list, and needs to put 'li' tag for each list items. like below:

  • NUMBER one
  • number 2

ol tag

Ol tag referes to ordered list and has the same format of using ul.
Since it is an ordered list, you can select a method of ordering by using an attribute 'type'
  1. cat
  2. dog
  3. oranges

dl tag

dl tag means definition list, has quite different tags.
Two different ones , 'dt' = definition title and 'dd' = definition data.
'dt' is used when defining a certain text and dd for explanations.
For example, menu of the restaurant.
