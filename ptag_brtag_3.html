Fun fact of p tag

1. p tag takes up the full one line of the web browser so as the size of te browser changes, it automatically changes the line and creates a new one. So depending on the size of the browser, the line of the given paragraph will differ.

2. No matter how many spaces you put inside the paragraph, it will consider as just one blank space. So when you enter and make bunch of lines inside the paragraph, won't work. So if you want to increase the space, use   "(&)nbsp(;)", (without the brackets) which is an entity code of HTML, which has the size of a space bar.(so extra space)
