One example would be create an "image" folder inside the folder where the html file are part of, then type in the image name. like above.
For more , going to add " ./ " in front of the image/--- as that refers to open the image folder from the current folder where the html file is running.

This is the way where the meaning gets completely different.
the " ../ " means the folder that contains the folder of html file is part of.
So an upper folder, to make it easy and since from that folder the image folder is not contained directly, indicating that the file route is wrong, so it'd give you an error.

So basically it is this :

HTML attribute : alt

In case the image won't load, use another attribute of "alt" to provide some explanations.
So the "alt" part shows up when the image cannot be displayed.

HTML attribute : width & height

this is an attribute that controls the size of the image.
If use either one of them, it will automatically control the size for you. So for example, if you only write width="300", the height will be automatically adjusted by the proportion of the orginial size of the pic.
If write both, it can damaage the actual proportion of the pic
Also can assign percentage to width and adjust the size.
